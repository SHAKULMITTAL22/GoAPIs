# GoAPIs

```sh
git clone https://github.com/kamerk22/GoAPIs
go run main.go
```

Basic CRUD APIs in Go with Gorilla-Mux. No database.

Router: [Gorilla-Mux](https://github.com/gorilla/mux) 

### APIs Implemented

```
GET     /api/books
GET     /api/books/{id} 
POST    /api/books/
PUT     /api/books/{id} 
DELETE  /api/books/{id} 
```
